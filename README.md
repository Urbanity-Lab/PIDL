# PIDL
Physics Informed Deep Learning - Traffic State Estimation

Data in this repo -     
1. Synthetic.mat - case study I - simulated traffic state dataset    
2. NGSIM_US80_4pm_Velocity_Data.txt - case study II - NGSIM Velocity Data
